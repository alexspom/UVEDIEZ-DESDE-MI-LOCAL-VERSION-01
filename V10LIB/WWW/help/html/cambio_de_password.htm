Cambio de password

Desde esta pantalla se podrá cambiar el password o contraseña para entrar en la aplicación.

Las teclas de función son las siguientes:

{F1} Ayuda {F4} Confirmar Cambios

 

Accede directamente a la tabla VDUSUARIO.

Los campos son los siguientes:

  • Usuario: Es el nombre del usuario que va a cambiar el password.

  • Password: Es el password actual del usuario.

  • Nuevo: Es el password nuevo del usuario.

  • Repita: Es el password nuevo del usuario que hay que volver a introducir por seguridad para comprobar que sea el mismo introducido en el campo anterior.
