Bultos de las zonas

Desde esta pantalla, se puede ver la información de los bultos en las zonas.

Las operaciones que se pueden realizar son latas, bajas o modificaciones. Es necesario el privilegio de “SUPERVISOR” para realizarlas.

Accede a la tabla VDBULTOZONA.

Los campos son:

  • Secuencia:Es el secuencial que tiene el bulto en la zona, es clave primaria y se obtiene del secuenciador VDSECBULTOZONA.

  • Zon: Es el código de la zona donde se encuentra el bulto.

  • Bulto: Es el código de bulto.

  • Codope: Es el código de operario que trata el bulto.

  • Sta.: Consta de dos campos, el primero es el estado del bulto y el segundo, su descripción.

  • Ultima modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
