Mantenimiento de almacenes

Accede directamente a la tabla VDALMA

Los campos que se tratan desde esta pantalla son los siguientes:

  • Cod: código que identifica al almacén y será único.

  • Denominación almacén: es el nombre o denominación del almacén.

  • Contacto: Es la persona de contacto de dicho almacén.

  • Dirección: es la dirección del almacén.

  • Población: la población dónde se encuentra el almacén.

  • País: consta de dos campos, el primero es el código y el Segundo el nombre del país del almacén. Con las teclas {May+F10} muestra lista de países para seleccionar, que es en modo consulta, el Mantenimiento de países y provincias

  • Provincia: consta de dos campos, el primero es el código y el Segundo el nombre de la provincia del almacén. Con las teclas {May+F10} muestra lista de provincias para seleccionar, que es en modo consulta, el Mantenimiento de provincias

  • D.P.: Es el distrito o código postal del almacén.

  • Teléfono1: es el número de teléfono del almacén.

  • Fax1: es el número de fax del almacén.

  • Teléfono2: es para introducir un Segundo número de teléfono, si lo hubiese.

  • Fax2: es para introducir un Segundo número de fax, si lo hubiese.

  • E-mail: es la dirección de correo electrónico del almacén.

  • EDI: Las siglas EDI corresponden al inglés Electronic Data Interchange (Intercambio Electrónico de Datos). EDI es el envío y recepción de documentos con medios telemáticos con el fin de posibilitar su tratamiento automático. Este sistema pretende que el emisor y receptor de un determinado documento comercial puedan ser directamente los respectivos ordenadores centrales, ganando así tiempo y evitando errores.

 

  • Última modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el Segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la modificación.
