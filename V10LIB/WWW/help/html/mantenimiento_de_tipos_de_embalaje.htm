Mantenimiento de tipos de embalaje

Desde esta pantalla se trata lo referente a los diferentes tipos de embalaje que utiliza la aplicación.

 

Las operaciones que se pueden realizar son altas, bajas y modificaciones, siempre y cuando se tenga el privilegio de “ADMINISTRADOR”.

 

Las teclas de función de estas pantallas son:

{F1} Ayuda {F4} Confirmar Cambios {F6} Cambio de pantalla o bloque

{F9} Borrar Registro {May-F10} Menú contextual;

 

Accede directamente a la tabla VDTIPOEMBA.

Los campos que se mantienen son:

 

  • Tipo: de dos campos, el primero es el tipo de embalaje de que se trata y el segundo es la designación del tipo de embalaje.

  • Codcomen: Código de comentario asociado el tipo de embalaje.

  • Última modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
