Mantenimiento de la equivalencia entre contenedores

Esta pantalla se utiliza para la comparativa de capacidades entre contenedores, se utiliza para calcular la altura de los contenedores.

Las operaciones que se pueden realizar en esta pantalla, son altas, bajas y modificaciones.

Las teclas de función son:

{F1} Ayuda {F2} Consulta {F4} Confirmar cambios {Esc} Cerrar Pantalla

{F9} Borrar un registro {F5} Borrar Pantalla

 

Accede directamente a la tabla VDCONTEEQ.

Los campos son:

  • Contenedor: consta de dos campos, el primero es el tipo de contenedor a comparar y el segundo su descripción.

  • Contenedor a comparar: consta de dos campos, el primero es el tipo de contenedor con el que se compara y el segundo su descripción.

  • Eq: Es el porcentaje del tamaño que es mayor(+), o que es menor(-)

  • Última modificación: Consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
