Reiniciar password

Desde esta pantalla, se podrá reiniciar un password a cualquier usuario, siempre y cuando se tenga el privilegio de “SUPERUSUARIO”

Las teclas de función son las siguientes:

{F1} Ayuda {F4} Confirmar Cambios

 

La pantalla no actúa sobre ninguna tabla de la aplicación.

Los campos son los siguientes:

  • Creación de password para usuario: Introduciremos el código del usuario al que se ha de crear un password..

  • Password: Es el nuevo password del usuario.

  • Repita: Es el password nuevo del usuario que hay que volver a introducir por seguridad para comprobar que sea el mismo introducido en el campo anterior.
