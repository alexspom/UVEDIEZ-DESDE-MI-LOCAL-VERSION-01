Mantenimiento de errores

Esta opción de menú permite monitorizar los diferentes módulos con los mensajes de error que tienen asignados.

Con el privilegio de "IMPLANTADOR", se podrán realizar las siguientes operaciones: altas, bajas y modificaciones.

Esta opción abre una pantalla que consta de dos bloques, uno referente a la tabla de módulos y el otro referente a los distintos mensajes de error de cada módulo.

Las operaciones permitidas desde ambos bloques son altas, bajas y modificaciones de los diferentes mensajes de mensajes de error de cada módulo.

Las teclas de función permitidas en estos bloques son:

{F1} Ayuda {F4} Confirmar Cambios {F6} Cambio de pantalla o bloque

{F9} Borrar Registro {May-F10} Menú contextual;
