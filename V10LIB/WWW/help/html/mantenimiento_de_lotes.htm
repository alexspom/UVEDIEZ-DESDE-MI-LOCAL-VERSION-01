Mantenimiento de lotes

Esta pantalla accede directamente a la tabla VDLOTES y permite el mantenimiento de los diferentes lotes que tiene un artículo.

Las operaciones que se pueden realizar desde ella son: altas, bajas y modificaciones, siempre y cuando se tenga el privilegio de “CALIDAD”.

Las teclas de función permitidas en este bloque son:

{F1} Ayuda {F4} Confirmar Cambios {F6} Cambio de pantalla o bloque

{F9} Borrar Registro {May+F10} Menú contextual

 

El menú contextual al que llama este bloque es VDLOTES y tiene varias opciones

Tiene los siguientes campos:

 

  • Artículo: es el código del artículo. Pulsando la tecla menú o la combinación de teclas {May+F10} nos muestra el menú VDARTIC

  • Designación: es la descripción correspondiente al código del campo artículo.

  • Lote: es el código del lote correspondiente al artículo.

  • Lote proveedor: es el código del lote del proveedor.

  • Caduci: es la fecha de caducidad del lote en formato ’.

  • Retest: es la fecha previa a la caducidad en la que se bloquea el lote. Tiene formato ’.

  • Bloqueos: son los bloqueos del lote. Tiene 10 indicadores, que según su valor indicarán una cosa u otra. Sus valores son:

    INDICADOR   VALOR  DESCRIPCION

 

1 Reservado   0  Libre Utilización 

2 Análisis   0  Libre Utilización
     1  Cuarentena
     2  Rechazado 

3 Precio   0  Precio Vigente
     1  Precio Antiguo
     2  Precio Nuevo
     3  Otro Precio 

4 Caducidad   0  Libre Utilización
 1  Precaducado
 2  Caducado 

5 Pendiente Acondicionar  0  Libre Utilización
    1  Pdte. Reencajar 

6 Recepción Pdte. Validar 0  Libre Utilización
     1  Pdte. Validar 

7 Indicador de Usuario  0  Libre utilización

     2  Posible Deterioro 

8 Disponible   0  Libre Utilización 

9 Disponible   0  Libre Utilización 

10 Disponible   0  Libre Utilización

 

  • Potencia: es la potencia o pureza del lote, va de 0 a 1, siendo este último el más puro. Se utiliza en fábrica.

  • Muestreado: Indica si se ha muestreado o no el lote.

  • Extra: es un campo de libre utilización, que contendrá información extra referente al lote.

  • Coment: es el código de comentario del contenedor.

  • Última modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
