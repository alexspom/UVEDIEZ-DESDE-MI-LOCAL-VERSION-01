Mantenimiento de secuenciales

Desde esta pantalla se mantienen los secuenciales de la aplicación. El secuencial concreto que se utiliza para la firma, no es un secuenciador de Oracle.

Con el privilegio de “IMPLANTADOR” se pueden realizar las operaciones de altas, bajas y modificaciones.

Las teclas de función que se pueden utilizar en esta pantalla son:

{F1} Ayuda {F4} Confirmar Cambios {F6} Cambio de pantalla

{F9} Borrar Registro  

 

Los campos que se mantienen desde esta pantalla son:

  • Nombre: Es el nombre del secuencial.

  • Valor: Es el valor actual del secuencial.

  • Inicio: Es el valor inicial del secuenciador

  • Máximo: Es el valor máximo que podrá alcanzar el secuencial.

  • Última modificación: Consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
