Mantenimiento de servicios

Desde este bloque se mantienen los servicios propiamente dichos.

Se necesita poseer el privilegio de “IMPLANTADOR” para poder realizar cualquier operación de alta, baja o modificación.

Las teclas de función son las siguientes:

{F1} Ayuda {F4} Confirmar Cambios {F6} Cambio de pantalla

{F9} Borrar Registro {May-F10} Menú contextual 

El menú contextual de esta pantalla es VDSERVICIOS

 

Accede a la tabla VDSERVICIOS.

Los campos son:

  • Servicio: Es el nombre del servicio perteneciente al grupo seleccionado en el bloque anterior.

  • Designación: es la designación del servicio.

  • A: Indica si el servicio se puede ejecutar.

  • Estado: consta de dos campos, el primero es el código del estado del servicio y el segundo su descripción.

  • PID:Es el número de identificador del programa, dentro del sistema operativo.

  • SID: Es el identificador de sesión del programa para Oracle.

  • SERIAL: Es el número de serie del programa para Oracle.

  • Ejecutable:Es el ejecutable que implementa el servicio.

  • Entorno: son las variables de entorno del servicio, separadas por (;) punto y coma.

  • Path V10: es la ruta padre o raíz particular del servicio, que se añade a la global del grupo.

  • Path: es la ruta particular del servicio, que se añade a la global del grupo.

  • Parámetros: Son los parámetros particulares del servicio, que se añaden a los globales del grupo.

  • Última modificación: Consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
