Mantenimiento de caminos

Desde este bloque se accede a la tabla VDCAMINOS, que representa las cabeceras de los caminos a seguir para alcanzar un destino desde un origen.

 

Tiene los siguientes campos:

 

  • Código: clave que identifica el camino.

  • Descripción: es la descripción del camino.

  • Área origen: es el código alfanumérico del área origen de este camino. Debe existir en el maestro de áreas. Pulsando May+F10nos muestra la pantalla de selección de área, para poder elegirla.

  • Área dest.: es el código alfanumérico del área destino de camino. Debe existir en el maestro de áreas. Pulsando May+F10nos muestra la pantalla de selección de área, para poder elegirla.

  • Última modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
