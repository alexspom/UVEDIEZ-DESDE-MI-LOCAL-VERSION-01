Mantenimiento de estados

Desde esta pantalla se mantienen los diferentes estados que intervienen en la aplicación.

Para poder realizar cualquier operación de tipo alta, baja o modificación, se ha de tener el privilegio de “IMPLANTADOR”.

Las teclas de función utilizadas en esta pantalla son:

{F1} Ayuda {F4} Confirmar Cambios {F9} Borrar Registro

 

Accede directamente a la tabla VDSTATUS.

Los campos de esta pantalla son:

  • Ti. Es el tipo de estado, con este campo se determina a que parte de la aplicación se hace referencia, por ejemplo MOV hará referencia a los movimientos.

  • Es.: es el valor o código del estado

  • Descripción: es la designación del estado.

  • Abreviatura: Es la forma abreviada de la designación, es imprescindible para la generación automática.

  • T: Se refiere al tipo de aplicación, sus valores son ’si es estándar y ’si es personalizada.

  • Última modificación: Consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
