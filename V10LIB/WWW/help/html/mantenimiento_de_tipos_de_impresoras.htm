Mantenimiento de tipos de impresoras

Esta pantalla consta de dos bloques, el primero es de tipos de impresoras en relación a lo que van a imprimir.

Para realizar operaciones de altas,baja o modificaciones, se necesita el privilegio de “ADMINISTRADOR”.

Las teclas de función son:

{F1} Ayuda {F4} Confirmar Cambios {F6} Cambio de pantalla

 

Accede a la tabla VDIMPRETIPO.

Los campos son:

  • Tipo: Es el tipo de impresión, por ejemplo etiquetas o informes.

  • Descripción: Es la descripción del tipo de impresión.

  • Activo: Indica si el tipo de impresión está activo. Sus valores (S)i o (N)o.

  • Tipo: consta de dos campos, el primero es el tipo de documento a imprimir, cuyo valor puede ser (I)nforme o (E)tiqueta. El segundo es la descripción del tipo.

  • Última modificación: Consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.

 

El segundo bloque es el Mantenimiento de asignación de informes
