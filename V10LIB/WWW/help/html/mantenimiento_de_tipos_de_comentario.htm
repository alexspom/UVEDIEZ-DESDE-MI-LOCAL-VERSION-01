Mantenimiento de tipos de comentario

Esta pantalla permite el mantenimiento de los comentarios de la aplicación.

 

Las operaciones que se pueden realizar desde esta pantalla son: altas, bajas y modificaciones con privilegios de “IMPLANTADOR”.

 

Las teclas de función permitidas en esta pantalla son:

{F1} Ayuda {F2} Consulta {F4} Confirmar cambios  {F5} Borrar pantalla

{F9} Borrar registro {Ctrl-F7} Copia {May-F10} Menú {Esc} Salir

 

accede directamente a la tabla ‘VDTIPOCOMEN’

Tiene los siguientes campos:

 

  • Ord.: indica el orden en el que se muestra el tipo de comentario.

  • Tabla: es la tabla a la que se asocia el tipo de comentario.

  • Tipo: indica el tipo de comentario.

  • Not.: indica si los eventos de este tipo de comentario deben notificarse automáticamente. Puede adoptar los valores S(i), N(o) o blanco.

  • Max. Líneas: indica el máximo de líneas de comentarios que puede tener un objeto de este tipo. El ‘’indica valor ilimitado.

  • Descripción: es la descripción del comentario.

  • Priv. Ver: indica el nivel de privilegio que tiene que tener el usuario para ver el comentario.

  • Priv. Modif.: indica el nivel de privilegio que tiene que tener el usuario para modificar un comentario.

  • Priv. Insertar: indica el nivel de privilegio que tiene que tener el usuario para insertar un comentario.

  • Última modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
