Mantenimiento del hardware de displays

Desde esta pantalla se mantiene la configuración del hardware de los displays de picking, para su correcto funcionamiento.

Las operaciones que se pueden realizar desde este bloque son altas y bajas, siempre que se posea el privilegio de “IMPLANTADOR.

 

Accede a la tabla VDINFOHARDDISP.

Los campos son:

 

  • Id.: Número de infopick hardware al que se refiere el display.

  • Numdisp: Número de display en la red InfoPick. Este código y el anterior forman la clave primaria.

  • Ge: Identificador del programa que gestiona la zona.

  • Texto: Es el texto que se muestra en el display.

  • Luces: Indica las luces que están encendidas en el display.

  • Teclas: Indica las teclas que están pulsadas en el display.

  • Status: Consta de dos campos, el primero es el estado del display y el segundo, su descripción.

  • Vdextra: Muestra información adicional del display.

  • Ultima modificación: consta de tres campos, el primero almacena quién realiza la modificación, bien sea un usuario o un procedimiento, el segundo y el tercero almacenan la fecha y la hora respectivamente, en la que se realizó la última modificación.
